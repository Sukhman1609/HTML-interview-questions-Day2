Explain br tag?

The <br> tag in HTML documents is used to create a line break in a text. It is an empty tag which means it does not need an end tag like </br> so we can move to next line.

Explain how many different types of lists we have in html

There are mainly three types of list in html ordered list, unordered list and description list i.e. ul,ol and dl. ul and ol have li inside it but dl contain dt and dd i.e.description term and details

What are semantic tags

A semantic tags clearly describes its meaning to both the browser and the developer and clearly defines its content. Examples: article, aside, details, figcaption, figure, footer, header, main, mark, nav, section, and summary. Examples of non-semantic elements: div and span - Tells nothing about its content.

Write code to create a link to https://www.prepbytes.com

    
To create links in HTML, we use anchor tags. These tags can be used to redirect
        to a new HTML page and also to a web page. So in href attribute we can give
        the web page link or we can give the location of the html file we want to load.
        target = “_blank” you can give when you want to open that page in a new tab.

        

        
Code:

        
            <h2>Link to go to Prepbytes website.</h2>
            <a href="https://www.prepbytes.com/" target="_blank">Prepbytes</a>
        

        
Output:

        
            
Link to go to Prepbytes website.

            Prepbytes
